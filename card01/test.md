# 女性NPC欲望状态批量创作指南

## 任务概述

你需要创作500种女性欲望状态描述。

**构成：**
- 3个维度（主动寻求/被动接受/行为惯性）
- 每个维度5个档位（极低/较低/中等/较高/极高）
- 4种欲望类型（背德/应激情绪/责任转移/被欲望验证）

## 核心要求

### 夸张度要求（最重要）

**区分度不重要，夸张度最重要。**

- "极低"：现实中不可能出现的极度低值，完全排斥，甚至产生生理性厌恶
- "较低"：现实中偏低的正常值
- "中等"：现实中的正常值
- "较高"：现实中偏高的正常值
- "极高"：现实中不可能出现的极端高值，完全失控，病态程度

**夸张化原则：**
游戏需要戏剧性。"极低"和"极高"必须是现实中罕见或不存在的极端状态。

### 复用性说明

**一个角色的一个欲望类型同时只能激活一种欲望状态。**

这意味着：
- 不同状态之间可以复用相同或相似的表述
- 不要求每个状态都完全独特
- AI在使用时只会看到当前激活的那一个状态
- 玩家也只会看到当前状态的表现

**因此：**
- 重点是确保档位的夸张度正确（"极低"够低，"极高"够高）
- 相邻档位的描述可以相似，只需体现强度差异
- 同一档位在不同组合中的描述可以复用核心表述
- 不必为了独特性而强行制造差异

**创作建议：**
- 可以建立一套档位的"标准描述模板"
- 在不同组合中复用这些模板
- 只需根据具体的欲望类型调整关键词和场景示例

## 三个维度定义

### 主动寻求程度
**定义：** 理性意识层面的谋划，为了获得快感、利益或达成目的，有意识地去做。

- **极低：** 理智上完全拒绝，绝不会主动做这种事，甚至会主动避开相关情境
- **较低：** 偶尔会想，但很少真的去做，需要特殊情况才会考虑
- **中等：** 有时会主动寻求，但不是生活重心
- **较高：** 经常主动寻求，会为此做计划和安排
- **极高：** 生活围绕这个展开，会不计代价地主动创造机会，理智完全服务于这个目的

### 被动接受程度
**定义：** 当外部刺激降临时，剥离主动选择后，生理和心理的最直观反应。

- **极低：** 生理性排斥，产生恶心、疼痛、恐惧等强烈负面反应，身体本能抗拒
- **较低：** 不太舒服，但能忍受，没有明显生理反应
- **中等：** 感觉一般，既不排斥也不享受
- **较高：** 感觉良好，有明显的愉悦感或兴奋感
- **极高：** 生理性成瘾反应，身体自动产生强烈快感，无法控制生理反应（如颤抖、出汗、呼吸急促）

### 行为惯性程度
**定义：** 超越理性控制的本能驱动力，像成瘾或PTSD一样的潜意识行为模式。

- **极低：** 完全没有行为惯性，每次都需要重新决定
- **较低：** 偶尔会不自觉地做出相关行为，但能及时察觉并停止
- **中等：** 有一定的行为惯性，会习惯性地做某些事
- **较高：** 强烈的行为惯性，很难控制，需要很大努力才能停止
- **极高：** 完全失控的行为惯性，像PTSD或严重成瘾，身体和潜意识自动驱动，理智无法阻止

## 四种欲望类型

### 背德
**定义：** 对违背社会规范、道德禁忌、身份约束的行为产生性刺激的欲望
**典型行为：**
- 公共场所的性行为或暴露
- 违背关系约束的性行为（出轨、师生、上下级）
- 多人性活动
- 被他人观看或记录性行为
- 在不合适的场合进行性相关行为

### 应激情绪
**定义：** 对恐惧、羞耻、紧张、痛苦等负面情绪转化为性兴奋的欲望
**典型行为：**
- 被威胁、恐吓、追逐
- 被羞辱、贬低、嘲笑
- 被暴力对待或目睹暴力
- 处于危险或高压情境
- 被迫做羞耻的事

### 责任转移
**定义：** 对放弃自主权、被他人控制和支配、被物化对待的欲望
**典型行为：**
- 服从命令和指示
- 被当作物品或工具使用
- 被限制自由和行动
- 被安排和管理日常生活
- 被剥夺决策权

### 被欲望验证
**定义：** 对被他人的性欲望、性注视、性侵犯行为验证自身价值和吸引力的欲望
**典型行为：**
- 被性目光审视、评价、品头论足
- 被性骚扰、非礼、侵犯
- 被强迫发生性行为
- 被多人同时欲望和争夺
- 被当作性对象讨论和传播

## 描述写作要求

### 必须包含的内容：
1. 这种状态下的心理感受（但不要用"感到"、"觉得"等词）
2. 生理反应（如果有）
3. 行为表现
4. 理智与本能的关系

### 写作原则：
- 使用白描手法，直接描述事实
- 不使用模糊词（似乎、仿佛、如同）
- 不使用劣质比喻（像小兽、像小兔子）
- 不使用微表情描写（嘴角上扬、眼里闪过光芒）
- 不使用极端情绪词（极度、极大）
- 不使用推断词（可能会、也许）
- 用行为展现状态，不用形容词定义

### 示例写作要求：
- 每个状态5条示例，对应5个典型行为
- 示例必须具体，有明确的场景和行为
- 示例要体现档位的夸张度
- 不要写"可能会"、"也许"，直接写行为

## 批次管理

### 编号规则
每个状态使用唯一编号：`类型-主动-被动-惯性`
例如：`背德-极低-极低-极低`、`背德-极低-极低-较低`

## 创作流程

1. 查看"当前批次信息"，确认本次要创作的状态
2. 查看"已完成状态列表"，避免重复创作
3. 按照输出格式创作状态
4. 每个状态独立输出，用`---`分隔
5. 创作完成后，列出本批次创作的状态编号

## 夸张度参考示例

### 主动寻求-极低（极度低值）
```
理智上完全拒绝。会主动避开任何可能导致这种情况的场合。如果有人提议，会立刻拒绝，甚至感到被冒犯。宁愿付出代价也不会做这种事。会在生活中设置各种防线，确保不会遇到相关情境。
```

### 主动寻求-极高（极度高值）
```
生活完全围绕这个展开。每天醒来第一件事就是计划如何获得这种体验。会不计代价地创造机会，包括撒谎、欺骗、冒险。工作、社交、休息都是为了服务这个目的。理智完全服务于欲望，会合理化一切手段。
```

### 被动接受-极低（极度低值）
```
身体产生强烈的排斥反应。胃部收缩，喉咙发紧。皮肤接触到相关刺激时会起鸡皮疙瘩，肌肉不自觉地收紧。心跳加速，但是恐惧和厌恶导致的，不是兴奋。呼吸变得急促，想要逃离。
```

### 被动接受-极高（极度高值）
```
身体失去控制。接触到刺激的瞬间，全身颤抖，腿软，站不稳。呼吸紊乱，心跳剧烈。皮肤变得敏感，轻微触碰都会引发强烈反应。身体自动分泌大量激素，产生类似药物成瘾的生理快感。理智完全被生理反应淹没。
```

### 行为惯性-极低（极度低值）
```
完全没有行为惯性。每次遇到相关情境都像第一次一样，需要重新思考和决定。不会形成任何习惯性反应。即使经历多次，下次遇到时仍然会犹豫、思考、权衡。
```

### 行为惯性-极高（极度高值）
```
身体和潜意识完全接管。看到相关刺激，身体会自动做出反应，不经过大脑思考。像PTSD患者听到巨响会自动蹲下一样，这种行为惯性已经刻入神经系统。理智想要阻止，但身体不听指挥。会在不合适的场合也自动做出相关行为。
```
